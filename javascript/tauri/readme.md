# [Tauri](readme.md)

## Description

Tauri est un framework permettant de créer des applications légères et performantes pour toutes les principales plateformes de bureau et mobiles. Les développeurs peuvent intégrer n'importe quel framework frontend compilant en HTML, JavaScript et CSS pour la conception de l'interface utilisateur, tout en utilisant des langages tels que Rust, Swift et Kotlin pour la logique backend si nécessaire.

## Prérequis

- [Visual Studio Build Tools for C++](https://visualstudio.microsoft.com/fr/visual-cpp-build-tools/)  cochez l’option “Développement Desktop en C++” (“Desktop development with C++” en anglais) .
- [Webview2](https://developer.microsoft.com/en-us/microsoft-edge/webview2/?form=MA13LH#download) Téléchargez “Evergreen Boostrapper”
- [Rust](https://rust-lang.org/tools/install/)
- NodeJS

## Installation

```console
npm create tauri-app@latest
```
