# [Nanocad](readme.md)

## Description

## Sortir du viewport

Utiliser la commande PSPACE

```console
ps
```

## Exporter toutes les pages

Utilise la commande `PUBLISH`. IL est préférable de n'avoir qu'un fichier ouvert.

## Faire pivoter un viewport

```console
UCS
Z (axe z)
90 (orientation)
```

```console
Plan
Current (c)
```

Ajuster la page

```console
zoom
Extend (e)
```
